Introduction Last updated: 2019-02-04

Habu - Agency Vuejs template based on vue-cli and vue-router.

Habu Vuejs template is a modern next generation template. It is created for business, corporate, company, personal, portfolio, startup, studio or any other similar use.
It is feature rich, responsive, easy to customize, and out of any jquery dependency. You can change any codes within if you have enough knowledge in HTML, CSS, Javascript and Vuejs coding.

Features Overview

  1. Made with Vue Cli
  2. Pure Vuejs, no jQuery
  3. Integrated with Vue-Router, no loading.
  4. Well structured and customizable CSS code.
  5. Fontawesome icons.
  6. Clean And Organized Code.
  7. Well Documented Code.
  8. Statndard Folder Stucture.
  9. And many more inside.....
